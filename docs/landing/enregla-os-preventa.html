Cómo funciona la pre-venta

  1. Reservas tu módulo (UAFE, SPDP, Societario…) como Founding Member.
  2. Aseguras precio de lanzamiento — el más bajo que va a existir.
  3. Acceso anticipado apenas el módulo esté listo; tu feedback define la prioridad.
  4. Mientras tanto, ya usas gratis todo el módulo de permisos.
