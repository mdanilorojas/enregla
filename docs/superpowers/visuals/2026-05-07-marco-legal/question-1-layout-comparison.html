Criterios de comparación

  • A — Grid de categorías Menos denso, obliga a 2 clicks (categoría → permiso). Categorías vacías se notan.
  • B — Chips + lista Ver todo de una. Filtro rápido. Búsqueda dominante. Un solo click al detalle.
  • C — Split view Estilo docs técnicas. Siempre ves categorías. Requiere más ancho de pantalla.
